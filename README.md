# Student Score Analyzer (Numpy)
This is a beginner friendly Numpy project that analyzes students scores:
## Concepts Used
- Numpy arrays
- np.stack
- axis operations
- np.where
- np.argsort
- np.unique
## Features
- Combines subject scores into a 2D array
- Calculates total and average scores
- Determines pass/fail status
- Sorts students by performance
## Technologies 
- Python
- Numpy
